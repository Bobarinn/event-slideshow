# Quick Setup Guide

## 1. Install Dependencies

```bash
npm install
```

## 2. Add Event Images

Place your images in `/public/event-images/` with these names:

- `1.jpg` or `1.png` - for Event #1 (Shoot Your First Short)
- `2.jpg` or `2.png` - for Event #2 (Campus Music Video Challenge)
- `3.jpg` or `3.png` - for Event #3 (Create-Your-Own Karaoke Video Night)
- ... and so on up to `25.jpg` or `25.png`

### Image Recommendations:
- **Size**: 1920x1080px (16:9) or 1200x1200px (square)
- **Format**: JPG or PNG
- **Quality**: High resolution for best presentation quality

## 3. Run Development Server

```bash
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000)
